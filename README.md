#  ECE 537: Image Processing and Computer Vision Spring 2026

This repository contains the code and resources for the ECE 537: Image Processing and Computer Vision course offered in 
Spring 2026. The course covers fundamental concepts and techniques in image processing and computer vision, including 
image enhancement, feature extraction, object recognition, and machine learning applications.

## Content Overview
- `assignments/`: Contains programming assignments and projects related to image processing and computer vision.
- `lectures/`: Lecture notes, slides, and supplementary materials for each class session.
